Chatbot WhatsApp

Bot de emprestimo/renegociacao (MVP)

Este MVP tem duas formas de uso:

- Terminal: fluxo simples no console.
- Web: interface com chat no navegador.

Como executar (web)

1) Instale o Node.js (>= 16).
2) Instale dependencias: 

npm install

3) Rode: 

npm start

Acesse http://localhost:3000.

Como executar (terminal)

node bot.js

Armazenamento

As conversas ficam em data/conversations.json.
